# Help App

Help is an app that displays emergency information for users that are registered in our database.

# Setting up

To run locally:

Check that you have node and npm installed.
```node -v

```

```npm -v

```
  

```git clone 
   cd root/Help-App
   npm install
   npm start
```
The app uses Node.js, Express, and MongoDB.
